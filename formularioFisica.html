Moto rettilineo uniforme:


Velocità media:


\(v_m = {\Delta S \over \Delta t}=\frac{S_1 –S_0}{t_1-t_0}\)


Legge oraria:


\(S(t) = S_0+ v (t-t_0)\)


Velocità media di due moti rettilinei uniformi con stessa distanza:


\(v_m = \frac{2}{{1\over v_1} +{1\over v_2} }\)


Velocità rettilineo di due moti rettilinei uniformi con stessa durata:


\(v_m = \frac{v_1 +v_2 }{2}\)


moto rettilineo uniformemente accelerato:


accelerazione media:


\(a_m =\frac{\Delta v}{\Delta t} = \frac{v_1 –v_0}{t_1 –t_0}\)


Legge oraria della velocità:


\(v(t) = v_0 + a(t-t_0)\)


Legge oraria:


\(S(t) = S_0 + v_0 (t-t_0) + {a (t-t_0)^2 \over 2}\)


Legge oraria nel caso \(t_0 =0\):


\(S(t) = S_0 + v_0 t + {at^2 \over 2}\)


Moto circolare uniforme:


velocità:


\(v=\frac{\Delta s}{\Delta t}= {2 \pi r \over T}\)


\(r=\)raggio


\(T=\)Periodo


Frequenza:


\(f= {1\over T}\)


velocità usando la frequenza:


\(v=2 \pi r f\)


velocità angolare:


\(w={\Delta \theta \over \Delta t}= {2 \pi \over T}\)


velocità usando la velocità angolare:


\(v=wr\)


accelerazione centripeta:


\(a_c = {v^2 \over r}\)


accelerazione centripeta usando la velocità angolare:


\(a_c = w^2 r\)

\(t_c =\sqrt {2h \over g}\)


velocità di caduta:


\(v_c = \sqrt {2hg}\)


tempo di salita:


\(t_s = t_c = \sqrt{2hg}\)


velocità di lancio:


\(|v_s| = |v_c|=\sqrt{2hg}\)


Moto sul piano inclinato:


velocità finale:


\(v_f=\sqrt{2hg}\)


tempo di caduta:


\(t_c = \sqrt{2h \over {g {\sin (\alpha) }^2}}\)


Moto armonico:


Ampiezza:


\(A=r\)


Pulsazione \(= w\)


Fase:


\(\phi = \theta_0\)


Legge oraria dell’angolo \(\theta\):


\(\theta= wt + \phi\)


legge oraria:


\(x(t) = A \cos (\theta)\)


\(x(t) = A \cos (wt + \phi)\)


Posizione massima:


\(x_{max} = A\)


velocità:


\(v=-wA\sin (\theta)\)


\(v=-wA \sin(wt +\phi)\)


velocità massima:


\(v_{max} = wA\)


accelerazione:


\(a=-w^2 A \cos (\theta)\)


\(a=-w^2 A\cos(wt +\phi)\)


accelerazione massima:


\(a_{max} = w^2 A\)
